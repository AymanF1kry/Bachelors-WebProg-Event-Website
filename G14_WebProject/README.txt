Steps on how to use Jom Lari 2024 website for Wampserver64:

1. Drop the G14_WebProject folder into the wamp64 -> www folder

2. In the MySQL folder, import the jomlari_db zip folder into phpMyAdmin server

3. Type at the search bar in your browser http://localhost/G14_WebProject/

4. In the Index of/G14_WebProject page, select jomlari_website/
